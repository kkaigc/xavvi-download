# Xavvi Downloads

Official Xavvi desktop client downloads for Windows and macOS.

## Latest Release

Download the current version from the [GitHub Releases](https://github.com/kkaigc/xavvi-download/releases/latest) page.

- Windows 11 x64: NSIS installer (`.exe`) or portable archive (`.zip`)
- macOS Apple Silicon: arm64 disk image (`.dmg`) or archive (`.zip`)
- macOS Intel: x64 disk image (`.dmg`) or archive (`.zip`)
- SHA-256 checksum files are provided with every release

This repository contains release downloads only. The application source code is maintained privately.
